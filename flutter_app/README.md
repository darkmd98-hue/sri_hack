# Skill Swap Flutter App

This is a starter Flutter client aligned with the backend API scaffold.

## Setup
1. Open this folder in your Flutter IDE:
   - `c:\flutter\skillswap\flutter_app`
2. Install dependencies:
   - `flutter pub get`
3. Update base URL if needed in:
   - `lib/core/app_config.dart`
4. Run:
   - `flutter run`

## Implemented Skeleton
- Auth flow (login/register + token persistence)
- Home (recommended matches)
- Explore (search teach listings)
- Requests (inbox/sent)
- Conversations + Chat (polling)
- Profile update
- Verification document upload trigger
